# TaskManager Frontend

React (JSX) + Vite frontend for the TaskManager Spring Boot API.

## Setup

1. Install dependencies:
   - `npm install`
2. Optional: copy `.env.example` to `.env` and adjust `VITE_API_URL`.
3. Run dev server:
   - `npm run dev`

The dev server proxies `/api` to `http://localhost:8080` by default.
